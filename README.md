개발 의도 

현재 제공되는 미세먼지 정보는 대부분 농도 중심의 단순 정보 제공에 그치고 있어, 실제 건강 위험도를 충분히 반영하지 못하는 한계가 있다. 특히 자치구별 인구 구조와 건강 상태의 차이를 고려하지 못한다는 점에서 지역 간 비교나 정책 판단에 활용하기에는 부족한 부분이 있다.

이러한 한계를 보완하기 위해 환경, 보건, 인구 데이터를 통합적으로 분석하여 지역별 보건 취약 특성을 보다 현실적으로 파악하고자 한다. 또한 분석 결과에 대한 원인을 함께 제시함으로써, 단순 정보 제공을 넘어 정책 의사결정에 활용 가능한 형태의 시스템을 구현하고자 한다.
